# Complete Installation Steps:
# 
# 1. Install System Dependencies (Windows):
#    - Visual Studio Build Tools 2019/2022
#    - Tesseract OCR
#    - Ollama
#
# 2. Create Virtual Environment:
#    python -m venv venv
#    venv\Scripts\activate  # Windows
#    source venv/bin/activate  # Linux
#
# 3. Upgrade pip:
#    python -m pip install --upgrade pip
#
# 4. Install Requirements:
#    pip install -r requirements.txt
#
# 5. Verify Installation:
#    python -c "import chromadb, ollama, flask; print('Success!')"
#
# 6. Pull LLM Model:
#    ollama pull llama3.2
#
# 7. Run Application:
#    python app.py
#
# Troubleshooting:
# ----------------
# - If cryptography fails: Install Visual C++ Build Tools
# - If pytesseract fails: Install Tesseract OCR and add to PATH
# - If chromadb fails: Ensure Python version is 3.10-3.12
